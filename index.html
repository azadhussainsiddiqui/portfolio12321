Next Basket Recommendation System


Next Basket Recommendation System helps predict what a customer is likely to buy next based on their past purchases and shopping history. It uses simple machine learning methods like finding items that are often bought together and studying buying patterns over time. The system looks at how products are related, how often they are bought, and the gap between purchases. It is commonly used in online shopping apps to give better product suggestions and improve customer experience. This project shows my skills in machine learning, data analysis, and building recommendation systems.

Job Application Tracker


Job Application Tracker is a system that helps users organize and manage their job applications in one place. It allows tracking of applied jobs, company details, application status, and interview updates. The system helps users stay organized and never miss follow-ups or deadlines. It can include features like reminders, filters, and status updates (applied, shortlisted, rejected). This project demonstrates skills in web development, database management, and user-focused design.
